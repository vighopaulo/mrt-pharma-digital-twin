# Phase 1.1 — Resource and Equipment Model

This increment extends the core domain layer with reusable hospital resources and major nuclear-medicine equipment.

## Added

- Generic resource model
- Cyclotron model
- Scanner model
- Clinical Cell model
- Resource cost calculations
- Scanner nominal capacity calculations
- Support for a zero-capacity cyclotron in a brand-new project
- Hospital, equipment, and resource unit tests
- Repository `.gitignore`

## Scope

This milestone defines planning-level infrastructure objects. It does not yet perform workflow simulation or operational dispatch.
